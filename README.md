# stocc-octo-asciitash

Interactive ASCII art that makes music. Move your mouse to create patterns and sound.

## How to Use

- **Click** anywhere to start
- **Click** to change patterns
- **Drag** to create melodies (X = pitch, Y = duration)

## Run Locally

```bash
git clone https://github.com/vyakart/stocc-octo-asciitash.git
cd stocc-octo-asciitash
npm install
npm start
```

## Deploy

```bash
npm run deploy
```

## Built With

React + Tone.js
